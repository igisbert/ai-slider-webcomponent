# AI Slider

Propuesta de componente web que muestra una comparativa visual de tipos de herramientas de IA, solicitado por [Manz.dev](https://manz.dev/).

El componente es adaptativo: renderiza los datos que tenga. Customizable mediante CSS Custom Properties.
